## Introduction

This portfolio is a personalized space designed to highlight my technical expertise, showcase my projects, and provide insights into my journey as an AI/ML Engineer and Cloud Professional. The website offers an interactive and user-friendly experience for visitors to explore my work, learn about my skills, and connect with me.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices and screen sizes

- **Modern Aesthetics**: Dark theme with vibrant purple-blue gradients and smooth animations

- **Animated Hero Section**: Eye-catching floating circles background with centered content

- **Project Showcase**: Detailed project cards featuring technologies used, descriptions, and live demo links

- **Skills Section**: Comprehensive display of technical skills organized by category

- **Achievements**: Highlight of certifications, academic achievements, and notable accomplishments

- **Contact Form**: Interactive contact form integrated with email for easy communication

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, animations, and transitions
- **JavaScript (ES6+)**: Interactive functionality and dynamic behavior
- **Font Awesome**: Icon library for social media and visual elements
- **Google Fonts (Inter)**: Modern, clean typography

## Project Structure

The project structure follows a standard static website layout. Key files include:

- `index.html`: Contains all HTML markup and sections
- `styles.css`: Complete styling including responsive design and animations
- `script.js`: Interactive features and scroll animations
